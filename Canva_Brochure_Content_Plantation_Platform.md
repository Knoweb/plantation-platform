- **Page 3 (Vision & Mission):** Optional faded background screenshot (`03`)

- **Page 10 (CTA):** Optional blurred screenshot background (`03`) + CTA text box

### D) Screenshot Quality Rules
- Use the same browser zoom (100%) for all captures
- Hide browser bookmarks and unrelated tabs
- Keep resolution at least 1920x1080 before cropping
- Crop to the product area only (remove desktop/taskbar)
- Avoid sensitive data (emails, phone numbers, internal IDs)
- Prefer data-rich states (charts, counts, pending badges visible)

### E) Quick Capture Workflow (Windows)
1. Open each screen in full browser window.
2. Use `Win + Shift + S` (Rectangular Snip).
3. Save with the naming format above.
4. Upload all images to Canva in one batch.
5. Apply one consistent frame style (same radius/shadow) across pages.
